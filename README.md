# 💱 Currency Converter (Java Console App)

A real-time currency converter built using Java.

## Features
- Fetches live exchange rates using API
- Displays available currency codes automatically
- Converts amount between currencies
- Exception handling for invalid inputs

## Technologies Used
- Java
- HttpURLConnection
- JSON (org.json)
- ExchangeRate API

## How to Run
1. Clone the repository
2. Add JSON library (org.json)
3. Run Main.java

## Sample Output
100 INR = 1.09 USD
